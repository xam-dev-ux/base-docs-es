# Base Docs - Español 🇪🇸

Traducción al español de la documentación de Agentes de IA de Base.

## 🌐 Ver la documentación

Este sitio está desplegado en Mintlify: [Próximamente]

## 📚 Contenido

- **Agentes de IA**: Documentación completa sobre cómo construir agentes de IA en Base
  - Frameworks (Agent SDK, OpenClaw, BANKR)
  - Wallets para agentes
  - Pagos y transacciones (x402)
  - Identidad, verificación y autenticación
  - Aplicaciones de agentes

## 🎯 Objetivo

Este proyecto es una traducción comunitaria de la documentación oficial de Base (https://docs.base.org/ai-agents) al español, con el objetivo de hacer la documentación más accesible para la creciente comunidad de desarrolladores de habla hispana en América Latina y España.

## 🤝 Contribución

Esta traducción fue creada siguiendo el mismo modelo que la traducción al coreano (https://daehan-base.mintlify.app/ai-agents-ko), en respuesta al llamado de @eric.base.eth de integrar traducciones comunitarias a los documentos oficiales de Base.

**Pull Request oficial**: https://github.com/base/docs/pull/1178

## 📝 Principios de traducción

- Términos técnicos mantenidos en inglés: wallet, token, SDK, API, x402, AgentKit, CDP, XMTP, onchain
- Todos los bloques de código preservados exactamente como en el original
- Tono profesional de documentación técnica
- "agent" → "agente", "identity" → "identidad", "payment" → "pago", "transaction" → "transacción"

## 🛠️ Tecnología

Este sitio utiliza [Mintlify](https://mintlify.com) para el despliegue y presentación de la documentación.
